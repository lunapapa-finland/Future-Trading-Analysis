# Agent Requirements

- Keep changes scoped to the requested task.
- Add or update tests for behavior changes.
- Run the relevant backend and frontend tests before handoff.
- Run CI/CD quality and security checks before handoff when code or dependencies change.
- Use and update the local Conda `finance_env` environment as project requirements change.
- Do not commit secrets, credentials, tokens, or private runtime data.
- Preserve existing user changes unless explicitly asked to revert them.
